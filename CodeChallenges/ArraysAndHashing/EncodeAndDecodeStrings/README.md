# Challenge: Encode and Decode Strings

## Source

Original problem: [Encode and Decode Strings](https://neetcode.io/problems/string-encode-and-decode/question)

## Problem Summary

This code requires two functions:
- The first will take a list of strings and encode them into a single string.
- The second will take a string and decode it to a list of strings.
The goal is for the list output by the decode function to be the same as the list input into the encode function.

## Assumptions and Constraints

Each of the strings to be encoded will contain only valid ASCII characters.

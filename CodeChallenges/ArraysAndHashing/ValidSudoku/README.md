# Challenge: Valid Sudoku

## Source

Original problem: [Valid Sudoku](https://neetcode.io/problems/valid-sudoku/question)

## Problem Summary

Check a standard 9 x 9 Sudoku board to see if it is valid.
To be valid, each row, column, and 3 x 3 block of the board must not contain any number from 1 to 9 more than once.

## Assumptions and Constraints

- The supplied board may not be complete.
- Cells that are not set to a number will be set to ``'.'``.

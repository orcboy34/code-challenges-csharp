# Challenge: Top K Frequent Elements

## Source

Original problem: [Top K Frequent Elements](https://neetcode.io/problems/top-k-elements-in-list/question)

## Problem Summary

Find the `k` most frequently occurring values in a given array of integers.

## Assumptions and Constraints

- The returned array can be in any order.
- `k` will be at least 1 and not greater than the number of distinct elements in the input array.

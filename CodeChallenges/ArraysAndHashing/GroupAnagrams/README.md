# Challenge: Group Anagrams

## Source

Original problem: [Group Anagrams](https://neetcode.io/problems/anagram-groups/question)

## Problem Summary

Group all strings in the given array of strings. The returned value should be an array of string arrays, with each inner array containing a group of words that are anagrams.

## Assumptions and Constraints

- The returned array can be in any order.
- All strings contain only lowercase English letters.
